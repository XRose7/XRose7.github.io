LeetCode 894 All Possible Full Binary Trees

Posted on 2024-04-02 In LeetCode

All Possible Full Binary Trees - LeetCode

Notes

Dynamic programming approach

动态规划 - 灵茶山艾府

\(f[i]\) 为有 \(i\) 个叶节点的所有真二叉树的列表. 左子树的所有真二叉树列表为 \(f[j]\) ，右子树的所有真二叉树列表为 \(f[i-j]\)

  • 时间复杂度：预处理 \(\mathcal{O}\left(\dfrac{4^N}{N^{3/2}}\right)\) ，其中 \(N=10\) 。根据卡特兰数，有 \(n\) 个叶子的真二叉树的个数为 \(C_{n-1} = \dfrac{(2n-2)!}{n!(n-1)!}\) 。其前 \(N\) 项之和为 \(\mathcal{O}\left(\dfrac{4^N}{N^{3/2}}\right)\)
  • 空间复杂度：预处理 \(\mathcal{O}\left(\dfrac{4^N}{N^{3/2}}\right)\) ，即 \(f\) 数组中的节点个数
