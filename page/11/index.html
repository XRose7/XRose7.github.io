LeetCode 525 Contiguous Array

Posted on 2024-03-16 In LeetCode

Contiguous Array - LeetCode

Notes

【宫水三叶】前缀和 + 哈希表

在预处理前缀和时，将 nums[i] 为 0 的值当做 -1 处理.

将问题转化为：如何求最长一段区间和为 0 的子数组. 我们可以使用 hash-table 记录某个前缀和出现的最小下标，将紧接着出现的前缀和出现的最小下标减去该值即可得到一段区间和为 0 的子数组长度.

Time: O(n)

Space: O(n)
