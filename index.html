LeetCode 2540 Minimum Common Value

Posted on 2024-03-09 In LeetCode

Minimum Common Value - LeetCode

Thinkings

类似于归并排序的双指针遍历比较. 当然也可以直接用 hash-table 遍历两数组，最后得出结果.

Codes

Time: O(n + m)

Space: O(1)

Notes

Binary search approach

由于两个数组都按升序排序，因此对于数组 \(\textit{nums}_1\)中的每个元素 \(\textit{num}\) ，可以使用二分查找判断 \(\textit{num}\) 是否在数组 \(\textit{nums}_2\) 中出现，在两个数组中都出现的最小元素即为答案。如果数组 \(\textit{nums}_1\) 中的所有元素都不在数组 \(\textit{nums}_2\) 中出现，则答案是 −1 .

由于两个数组都按升序排序，因此可以从左向右遍历数组 \(\textit{nums}_1\) 中的每个元素，找到的第一个在两个数组中都出现的元素即为答案.

Time: O(mlogn)

Space: O(1)

LeetCode 2575 Find the Divisibility Array of a String

Posted on 2024-03-07 In LeetCode

Find the Divisibility Array of a String - LeetCode

Thinkings

找出字符串的可整除数组 - 力扣官方题解

我们可以将主要问题抽象为计算 \(a \; \text{mod} \; m\) 以及 \((a \times 10 + b) \; \text{mod} \; m\) ，而 \[ (a \times 10 + b) \; \text{mod} \; m = (a \; \text{mod} \; m \times 10 + b) \; \text{mod} \; m \] 由此递推式可解.

Codes

Time: O(n)

Space: O(1) 不计算返回结果

Notes

Quick multiply approach

可先参阅：

快速幂 - OI Wiki

数据直接乘 10 可能会爆 int ，要是不想爆，可以将乘法改成加法，同时每一步都 mod m . 而加法还可以使用快速乘算法，将被乘数写成二进制形式，再用乘法分配律将其写成加法形式.

快速乘会比普通的循环加法快一些，但花费时间还是比直接使用 long long 类型多.

Time: O(n)

Space: O(1) 不计算返回结果.
